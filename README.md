## Installation

Use the package manager [npm](https://www.npmjs.com/) to install React-Parallax-Animation.

```bash
npm install
```

## Usage

```javascript
import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';



ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById('root')
);

```
